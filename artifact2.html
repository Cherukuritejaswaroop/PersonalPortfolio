Artifact 2

EcoTravel AI - Sustainable Travel Chatbot

This artifact presents an AI-powered travel chatbot designed to recommend eco-friendly and sustainable travel choices.

Live Demo

Artifact Information

EcoTravel AI demonstrates how Generative AI can turn travel intent into practical recommendations that prioritize responsible tourism, greener transportation, and lower-impact planning.

Introduction

This artifact explores how conversational AI can support sustainability by giving users more accessible, guided travel planning choices.

Description

The chatbot was designed to respond to user travel goals while keeping environmental responsibility visible inside the planning process.

It combines prompt engineering, structured response design, and design-thinking ideas to produce outputs that are practical, clear, and user friendly.

Objective

  • Generate eco-friendly travel recommendations
  • Improve relevance through prompt engineering
  • Design a more practical sustainability-focused chatbot flow

Process

  • Identified sustainable tourism scenarios and traveler goals
  • Designed prompts to improve recommendation quality and clarity
  • Applied design thinking from empathy through testing
  • Refined responses to make outputs easier to use and compare

Tools and Technologies Used

Generative AI LLM Prompt Engineering Design Thinking

Value Proposition

This artifact shows how AI can transform open-ended user requests into practical travel suggestions with a clear sustainability focus.

Unique Value: Helps users explore better travel choices through guided eco-aware responses.

Relevance: Useful for understanding AI-assisted planning and user-centered response design.

Reflection

This project improved my understanding of how prompt structure, response clarity, and user context affect the usefulness of AI-generated outputs.

References

OpenAI GPT resources; prompt engineering concepts; sustainable tourism and travel-planning guidance.
